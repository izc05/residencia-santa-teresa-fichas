RESIDENCIA SANTA TERESA

Ficha editable instalable para generar PDF.

USO LOCAL
1. Abre index.html en Chrome, Edge o Safari.
2. Introduce la contraseña inicial: SantaTeresa2026!
3. Rellena los campos.
4. Pulsa "Generar PDF".
5. Guarda también una copia con "Exportar editable" para poder modificarla más tarde.

INSTALAR COMO APP
- En Chrome o Edge: abre la app y pulsa "Instalar app".
- En Android: abre la URL y usa "Instalar app" o "Añadir a pantalla de inicio".
- En iPad/iPhone: abre la URL con Safari y usa Compartir > Añadir a pantalla de inicio.

PUBLICAR EN GITHUB PAGES
1. Sube esta carpeta a un repositorio de GitHub.
2. En GitHub, entra en Settings > Pages.
3. Elige la rama principal y la carpeta root.
4. GitHub generará una URL pública.
5. Abre esa URL en Chrome/Edge para instalar la app.

IMPORTANTE SOBRE LA CONTRASEÑA
La pantalla de contraseña es una barrera básica para una app estática. Si se van a guardar datos sensibles o reales de residentes en internet, debe usarse un sistema con autenticación real de servidor o un repositorio/hosting privado.
